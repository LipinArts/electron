WHAT IS ELECTRON?

  • Open source library for creating cross-platform desktop applications with HTML, CSS and JS
  • Combines Chromium and Node.js into single runtime
  • Apps can be packaged for Windows, MacOS and Linux

History of Electron

  • Project began in January 2013 and formally knows as Atom shell (originally was written for Atom)
  • Project was open sourced in May of 2014
  • Renamed to Electron in April of 2015
  • Released version 1.0.0 in June of 2016

It has since become a popular tool used by open source developers, startups and established companies

Electron under the hood

  • Main Process - controls the life of the application. Creates Browser Window instances to run web page.
  • Render Process - is a browser window in your app. Each window runs its own isolated process.

A minimal Electron application

Electron structure

Package.json

{
  "name": "new_app",
  "version": "1.7.9",
  "main": "main.js",
  "scripts": {
    "start": "electron ."
    }
}

Main.js

const {app, BrowserWindow} = require('electron')
let mainWindow = null;
app.on('ready', () => {
  mainWindow = new BrowserWindow({ width: 500, height: 400 });
  mainWindow.loadURL('file://' + __dirname + '/index.html');
});

index.html

<html>
  <head>
    <script>
      window.onload = () => {
        const fs = require('fs');
        const p = document.createElement('p');
        p.textContent = fs.readFileSync('cats.txt');
        document.body.appendChild(p);
      }
    </script>
  </head>
  <body>
    <h1>Electron is awesome.</h1>
  </body>
</html>

Benefits of using Electron

  • Electron uses Web tools to create apps.
  • Thanks to a simplified, single-shared library, users can build Electron app without a powerful machine.
  • Applications built with Electron can be run on most platforms, including Mac, Linux and Windows.
  • Electron applications run the same regardless of the operating system.
